Pandas Programming

1. Pandas HOME

Pandas is a fast, powerful, flexible, and easy-to-use open-source data analysis and manipulation library for Python.

import pandas as pd
data = {'Name': ['Tom', 'Nick', 'John'], 'Age': [20, 21, 19]}
df = pd.DataFrame(data)
print(df) 
                   Name  Age
                0   Tom   20
                1  Nick   21
                2  John   19

2. Pandas Intro

Pandas helps to manipulate, analyze and clean the data in a simple manner. It provides two main structures: Series and DataFrame.

import pandas as pd
data = {'Product': ['A', 'B', 'C'], 'Price': [10, 20, 30]}
df = pd.DataFrame(data)
print(df) 
                   Product  Price
                0       A     10
                1       B     20
                2       C     30

4. Pandas Series

A Series is a one-dimensional array-like object that holds data of any type.

import pandas as pd
data = [1, 2, 3, 4, 5]
series = pd.Series(data)
print(series) 
                0    1
                1    2
                2    3
                3    4
                4    5
                dtype: int64

5. Pandas DataFrames

DataFrame is a two-dimensional size-mutable, potentially heterogeneous tabular data structure.

import pandas as pd
data = {'Name': ['John', 'Doe'], 'Age': [23, 29]}
df = pd.DataFrame(data)
print(df) 
                   Name  Age
                0  John   23
                1   Doe   29

8. Pandas Analyzing Data

Use pandas to analyze data easily, such as calculating the mean, sum, and other statistics.

import pandas as pd
data = {'Name': ['Alice', 'Bob', 'Carol'], 'Age': [30, 25, 22]}
df = pd.DataFrame(data)
print(df.mean()) 
                Age    25.666667
                dtype: float64

9. Cleaning Data

Data cleaning involves removing or fixing bad data.

import pandas as pd
data = {'Name': ['Alice', 'Bob', None], 'Age': [30, None, 25]}
df = pd.DataFrame(data)
df = df.dropna()
print(df) 
                   Name   Age
                0  Alice  30.0

10. Cleaning Empty Cells

Remove empty cells (NaN) using dropna() or fill them with a specific value.

import pandas as pd
data = {'Name': ['Alice', None, 'Carol'], 'Age': [30, 25, None]}
df = pd.DataFrame(data)
df.fillna('Unknown', inplace=True)
print(df) 
                   Name     Age
                0  Alice     30.0
                1  Unknown  25.0
                2  Carol    Unknown

11. Cleaning Wrong Format

Fix incorrect data formats such as date formatting.

import pandas as pd
data = {'Date': ['2025-01-01', '2025-01-02']}
df = pd.DataFrame(data)
df['Date'] = pd.to_datetime(df['Date'])
print(df) 
                   Date
                0 2025-01-01
                1 2025-01-02

12. Cleaning Wrong Data

Remove rows with incorrect data based on certain conditions.

import pandas as pd
data = {'Age': [25, -30, 22]}
df = pd.DataFrame(data)
df = df[df['Age'] > 0]
print(df) 
                   Age
                0   25
                2   22

13. Removing Duplicates

Remove duplicate rows in the data.

import pandas as pd
data = {'Name': ['Alice', 'Bob', 'Alice'], 'Age': [30, 25, 30]}
df = pd.DataFrame(data)
df = df.drop_duplicates()
print(df) 
                   Name   Age
                0  Alice  30
                1   Bob   25

14. Pandas Correlations

Find correlations between numerical columns using pandas.

import pandas as pd
data = {'A': [1, 2, 3], 'B': [4, 5, 6]}
df = pd.DataFrame(data)
print(df.corr()) 
                   A    B
                A  1.0  1.0
                B  1.0  1.0

15. Pandas Plotting

Pandas allows you to plot data directly from DataFrames or Series using the `.plot()` method. It is built on Matplotlib.

import pandas as pd
import matplotlib.pyplot as plt
data = {'x': [1, 2, 3, 4, 5], 'y': [5, 4, 3, 2, 1]}
df = pd.DataFrame(data)
df.plot(x='x', y='y', kind='line')
plt.show()

(A line plot will be displayed with the given data)

16. Pandas Plotting

Creating different types of plots with pandas' built-in plotting functionalities.

import pandas as pd
data = {'x': [1, 2, 3, 4, 5], 'y': [2, 4, 6, 8, 10]}
df = pd.DataFrame(data)
df.plot(kind='bar')
plt.show()

(A bar plot will be displayed based on the given data)
