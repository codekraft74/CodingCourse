2. TensorFlow Introduction

TensorFlow is an open-source machine learning library developed by Google. It allows you to build and train machine learning models using dataflow graphs.

3. TensorFlow - Installation

To install TensorFlow, use pip to install it from the command line:

pip install tensorflow

Artificial Intelligence (AI) refers to the simulation of human intelligence in machines. It involves tasks such as learning, reasoning, problem-solving, and understanding.

Linear algebra helps in representing data as matrices and vectors, while calculus helps optimize functions during model training.

6. Machine Learning & Deep Learning

Machine learning is a subset of AI where models learn from data without being explicitly programmed. Deep learning, a subset of machine learning, uses neural networks to model complex patterns in data.

7. TensorFlow - Basics

TensorFlow uses tensors (multi-dimensional arrays) as its core data structure. Operations on tensors can be expressed as graphs, and TensorFlow handles these operations efficiently.

import tensorflow as tf
tensor = tf.constant([1, 2, 3, 4])
print(tensor)

8. Convolutional Neural Networks (CNNs)

Convolutional Neural Networks (CNNs) are a type of deep learning model commonly used for image classification and computer vision tasks. They use convolutional layers to extract features from images.

from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D
model = Sequential([
  Conv2D(32, (3, 3), activation='relu', input_shape=(64, 64, 3)),
  MaxPooling2D((2, 2))
]) 
Model Summary:
Model: "sequential"
        Layers:
         - Conv2D (32 filters, 3x3 kernel, ReLU activation)
         - MaxPooling2D (2x2 pooling)

9. Recurrent Neural Networks (RNNs)

Recurrent Neural Networks (RNNs) are a class of neural networks that are particularly effective for sequential data, such as time series or text data.

from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import SimpleRNN
model = Sequential([
  SimpleRNN(50, input_shape=(100, 1))
]) 
Model Summary:
Model: "sequential"
        Layers:
         - SimpleRNN (50 units, input shape (100, 1))

10. TensorBoard Visualization

TensorBoard is a visualization tool that helps in tracking and visualizing the training process of a model, including metrics like loss, accuracy, and more.

from tensorflow.keras.callbacks import TensorBoard
tensorboard = TensorBoard(log_dir='./logs')
model.fit(X_train, y_train, epochs=10, callbacks=[tensorboard])

11. Single Layer Perceptron

A Single Layer Perceptron (SLP) is the simplest type of neural network with only one layer of neurons. It is a linear classifier used for binary classification tasks.

from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense
model = Sequential([
  Dense(1, activation='sigmoid', input_shape=(2,))
]) 
Model Summary:
Model: "sequential"
        Layers:
         - Dense (1 unit, sigmoid activation)

model = tf.keras.Sequential([

TFLearn is a Python library built on top of TensorFlow for simplifying machine learning model creation. To install TFLearn, use pip:

14. CNN and RNN Difference

Convolutional Neural Networks (CNNs) are used for image-related tasks, as they are good at capturing spatial hierarchies in data. Recurrent Neural Networks (RNNs), on the other hand, are used for sequential data like time series or text, as they maintain memory across time steps.

15. TensorFlow - Keras

Keras is an open-source neural network library written in Python. It runs on top of TensorFlow and allows for fast prototyping and easy model creation.

from tensorflow import keras
model = keras.Sequential([
  keras.layers.Dense(128, activation='relu', input_shape=(784,)),
  keras.layers.Dense(10, activation='softmax')
]) 
Model Summary:
Model: "sequential"
        Layers:
         - Dense (128 units, ReLU activation)
         - Dense (10 units, Softmax activation)

16. TensorFlow Distributed Computing

TensorFlow supports distributed computing to scale your models and training processes. You can use TensorFlow's `tf.distribute.Strategy` to distribute training across multiple devices or machines.

strategy = tf.distribute.MirroredStrategy()
with strategy.scope():
  model = tf.keras.Sequential([...])
  model.fit(x_train, y_train) 
Distributed training started using MirroredStrategy.

17. TensorFlow - Exporting

TensorFlow allows you to export your trained models for deployment. You can save and restore models using the `SavedModel` format.

model.save("model_path") 
Model saved successfully to model_path.

18. Multi-Layer Perceptron Learning

Multi-layer Perceptrons (MLPs) are neural networks with multiple layers, including hidden layers. They can model complex relationships between inputs and outputs, and are typically used for classification tasks.

from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense
model = Sequential([
  Dense(64, activation='relu', input_shape=(784,)),
  Dense(10, activation='softmax')
]) 
Model Summary:
Model: "sequential"
        Layers:
         - Dense (64 units, ReLU activation)
         - Dense (10 units, Softmax activation)

19. Hidden Layers of Perceptron

Hidden layers in a perceptron are layers that are between the input and output layers. They help the model learn more complex patterns by adding non-linearity to the model.

model = Sequential([
  Dense(64, activation='relu', input_shape=(784,)),
  Dense(32, activation='relu'),
  Dense(10, activation='softmax')
]) 
Model Summary:
Model: "sequential"
        Layers:
         - Dense (64 units, ReLU activation)
         - Dense (32 units, ReLU activation)
         - Dense (10 units, Softmax activation)

20. TensorFlow - Optimizers

Optimizers are algorithms used to minimize the loss function during training by adjusting the weights of the model. Some common optimizers are Adam, SGD, and RMSProp.

model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy']) 
Optimizer set to Adam. Model is ready for training.

21. TensorFlow - XOR Implementation

Implementing the XOR function with TensorFlow involves using a neural network to learn the logic behind XOR, a non-linear operation. This can be achieved with a multi-layer perceptron.

import tensorflow as tf
model = tf.keras.Sequential([
  tf.keras.layers.Dense(4, input_dim=2, activation='relu'),
  tf.keras.layers.Dense(1, activation='sigmoid')
])
model.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])
model.fit(x, y, epochs=100) 
Model trained to learn the XOR function.

22. Gradient Descent Optimization

Gradient descent is an optimization algorithm used to minimize the loss function by updating the weights of the neural network based on the gradient of the loss.

optimizer = tf.keras.optimizers.SGD(learning_rate=0.01)
model.compile(optimizer=optimizer, loss='mean_squared_error', metrics=['accuracy']) 
Optimizer set to Stochastic Gradient Descent (SGD).

23. TensorFlow - Forming Graphs

In TensorFlow, computation is described in the form of a computational graph. This graph contains nodes representing operations and edges representing data dependencies between operations.

import tensorflow as tf
node1 = tf.constant(3)
node2 = tf.constant(4)
sum = node1 + node2
with tf.Session() as sess:
  print(sess.run(sum))

24. Image Recognition using TensorFlow

TensorFlow can be used to create models for image recognition. The typical workflow involves using Convolutional Neural Networks (CNNs) to identify patterns in images.

model = tf.keras.Sequential([
  tf.keras.layers.Conv2D(32, (3, 3), activation='relu', input_shape=(64, 64, 3)),
  tf.keras.layers.MaxPooling2D(pool_size=(2, 2)),
  tf.keras.layers.Flatten(),
  tf.keras.layers.Dense(10, activation='softmax')
])
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy']) 
Model ready for image recognition tasks.
