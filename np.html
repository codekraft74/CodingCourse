NumPy Programming

1. NumPy HOME

Welcome to the NumPy tutorial. NumPy is a Python library used for numerical computing and working with arrays. It is widely used in data science, machine learning, and scientific computations.

(Introduction to NumPy and its importance)

2. NumPy Intro

NumPy stands for Numerical Python. It provides support for multi-dimensional arrays and matrices, along with mathematical operations to work on these arrays efficiently.

import numpy as np
arr = np.array([1, 2, 3, 4, 5])
print(arr) 
[1 2 3 4 5]

3. NumPy Getting Started

Before using NumPy, ensure it is installed. Install NumPy using pip:

pip install numpy

Create your first NumPy array:

import numpy as np
arr = np.array([10, 20, 30])
print(arr) 
[10 20 30]

4. NumPy Creating Arrays

Create arrays using NumPy's `array()` function.

import numpy as np
arr = np.array([1, 2, 3, 4, 5])
print(arr) 
[1 2 3 4 5]

5. NumPy Array Indexing

Access elements of an array using indexing.

import numpy as np
arr = np.array([10, 20, 30, 40, 50])
print(arr[1]) 
20

6. NumPy Array Slicing

Slicing allows you to extract a portion of an array.

import numpy as np
arr = np.array([10, 20, 30, 40, 50])
print(arr[1:4]) 
[20 30 40]

7. NumPy Data Types

Every NumPy array has a data type, like `int`, `float`, or `string`.

import numpy as np
arr = np.array([1, 2, 3], dtype='float')
print(arr) 
[1. 2. 3.]

8. NumPy Copy vs View

Understand the difference between copying an array and creating a view.

import numpy as np
arr = np.array([1, 2, 3])
copy_arr = arr.copy()
view_arr = arr.view()
arr[0] = 10
print(copy_arr)
print(view_arr) 
        [1 2 3]
        [10 2 3]

9. NumPy Array Shape

Check the shape of an array using the `shape` attribute.

import numpy as np
arr = np.array([[1, 2, 3], [4, 5, 6]])
print(arr.shape) 
(2, 3)

10. NumPy Array Reshape

Change the shape of an array using the `reshape()` method.

import numpy as np
arr = np.array([1, 2, 3, 4, 5, 6])
new_arr = arr.reshape(2, 3)
print(new_arr) 
        [[1 2 3]
         [4 5 6]]

16. NumPy Random

Generate random numbers using the `random` module in NumPy.

import numpy as np
random_num = np.random.rand(5)
print(random_num) 
[0.123 0.456 0.789 0.101 0.112]

17. Random Intro

The random module in NumPy provides functions for generating random numbers, arrays, and distributions.

import numpy as np
arr = np.random.randint(0, 10, size=(3, 3))
print(arr) 
        [[1 4 7]
         [2 5 8]
         [3 6 9]]

18. Data Distribution

Data distribution is a function that defines the likelihood of different outcomes in a dataset. NumPy provides tools to generate and analyze various distributions.

import numpy as np
data = np.random.normal(0, 1, 1000)
print(data[:10]) 
[0.1, -0.3, 1.2, -0.5, 0.9, 0.4, -1.1, 0.7, -0.8, 1.5]

19. Random Permutation

A permutation randomly changes the order of elements in an array.

arr = np.array([1, 2, 3, 4, 5])
permuted = np.random.permutation(arr)
print(permuted) 
[3, 5, 1, 4, 2]

20. Seaborn Module

Seaborn is a library used for creating statistical graphics. You can visualize distributions generated with NumPy.

import seaborn as sns
import matplotlib.pyplot as plt
data = np.random.normal(0, 1, 1000)
sns.histplot(data, kde=True)
plt.show()

(Displays a histogram with a density curve)

21. Normal Distribution

A normal distribution is symmetric and describes many natural phenomena.

data = np.random.normal(5, 2, 1000)
print(data[:10]) 
[5.2, 3.8, 4.9, 6.1, 5.4, 3.7, 6.3, 5.0, 4.2, 6.5]

22. Binomial Distribution

The binomial distribution describes the number of successes in a sequence of independent experiments.

data = np.random.binomial(n=10, p=0.5, size=10)
print(data) 
[5, 7, 4, 6, 5, 3, 7, 6, 5, 4]

23. Poisson Distribution

The Poisson distribution describes the probability of events occurring within a fixed interval of time or space.

data = np.random.poisson(3, 10)
print(data) 
[2, 4, 3, 5, 3, 2, 1, 4, 3, 6]

30. NumPy ufunc

Universal functions (ufuncs) are used to perform element-wise operations on arrays.

31. ufunc Intro

A ufunc (universal function) operates on ndarrays element-wise, supporting broadcasting.

arr = np.array([1, 2, 3, 4])
result = np.add(arr, 5)
print(result) 
[6, 7, 8, 9]

32. ufunc Create Function

Create your own ufunc using `frompyfunc`.

def multiply(x, y):
   return x * y
ufunc = np.frompyfunc(multiply, 2, 1)
result = ufunc([1, 2, 3], [4, 5, 6])
print(result) 
[4, 10, 18]

33. ufunc Simple Arithmetic

Use ufuncs for basic arithmetic operations.

arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])
result = np.add(arr1, arr2)
print(result) 
[5, 7, 9]

34. ufunc Rounding Decimals

Round elements of an array to a specified number of decimals.

arr = np.array([1.234, 5.678, 9.1011])
result = np.round(arr, 2)
print(result) 
[1.23, 5.68, 9.1]

35. ufunc Logs

Calculate the natural logarithm of each element in an array using `np.log`.

arr = np.array([1, 2, 3, 4])
result = np.log(arr)
print(result) 
[0.0, 0.693, 1.099, 1.386]

36. ufunc Summations

Use NumPy's ufuncs to calculate the sum of elements in an array.

arr = np.array([1, 2, 3, 4, 5])
result = np.sum(arr)
print(result) 
15

37. ufunc Products

Calculate the product of all elements in an array using `np.prod`.

arr = np.array([1, 2, 3, 4])
result = np.prod(arr)
print(result) 
24

38. ufunc Differences

Find the differences between consecutive elements in an array.

arr = np.array([10, 20, 30, 40])
result = np.diff(arr)
print(result) 
[10, 10, 10]

39. ufunc Finding LCM

Find the Least Common Multiple (LCM) of numbers in an array.

arr = np.array([4, 6, 8])
result = np.lcm.reduce(arr)
print(result) 
24

40. ufunc Finding GCD

Find the Greatest Common Divisor (GCD) of numbers in an array.

arr = np.array([8, 12, 16])
result = np.gcd.reduce(arr)
print(result) 
4

41. ufunc Trigonometric

Perform trigonometric calculations like sine, cosine, and tangent using NumPy ufuncs.

angles = np.array([0, np.pi/2, np.pi])
result = np.sin(angles)
print(result) 
[0.0, 1.0, 0.0]

42. ufunc Hyperbolic

Perform hyperbolic calculations like sinh, cosh, and tanh.

values = np.array([1, 2, 3])
result = np.sinh(values)
print(result) 
[1.175, 3.627, 10.018]

43. ufunc Set Operations

Use NumPy for set operations like union, intersection, and difference.

set1 = np.array([1, 2, 3, 4])
set2 = np.array([3, 4, 5, 6])
union = np.union1d(set1, set2)
intersection = np.intersect1d(set1, set2)
difference = np.setdiff1d(set1, set2)
print("Union:", union)
print("Intersection:", intersection)
print("Difference:", difference) 
        Union: [1, 2, 3, 4, 5, 6]
        Intersection: [3, 4]
        Difference: [1, 2]
