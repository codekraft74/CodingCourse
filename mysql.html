2. MySQL Intro

MySQL is an open-source relational database management system...

3. MySQL RDBMS

RDBMS stands for Relational Database Management System...

4. MySQL SQL

SQL stands for Structured Query Language. It is used to communicate with databases...

5. MySQL SELECT

The SELECT statement is used to select data from a database...

SELECT * FROM Students;
StudentID StudentName Age
1 John 22
2 Jane 20

6. MySQL WHERE

The WHERE clause is used to filter records...

SELECT * FROM Students WHERE Age = 22;
StudentID StudentName Age
1 John 22

7. MySQL AND, OR, NOT

AND, OR, and NOT are logical operators used to filter records based on multiple conditions...

SELECT * FROM Students WHERE Age > 18 AND Age < 25;
StudentID StudentName Age
1 John 22
2 Jane 20

8. MySQL ORDER BY

The ORDER BY clause is used to sort the result-set in ascending or descending order...

SELECT * FROM Students ORDER BY Age DESC;
StudentID StudentName Age
1 John 22
2 Jane 20

9. MySQL INSERT INTO

The INSERT INTO statement is used to insert new records into a table...

INSERT INTO Students (StudentName, Age) VALUES ('Alice', 23);

10. MySQL NULL Values

NULL represents a missing value in a database...

SELECT * FROM Students WHERE Age IS NULL;

No records found where Age is NULL.

11. MySQL UPDATE

The UPDATE statement is used to modify existing records in a table...

UPDATE Students SET Age = 24 WHERE StudentID = 1;

12. MySQL DELETE

The DELETE statement is used to remove records from a table...

DELETE FROM Students WHERE Age < 20;

13. MySQL LIMIT

The LIMIT clause is used to specify the number of records to return...

SELECT * FROM Students LIMIT 3;

14. MySQL MIN and MAX

The MIN() function returns the smallest value, and MAX() returns the largest value...

SELECT MIN(Age) AS MinimumAge, MAX(Age) AS MaximumAge FROM Students;

15. MySQL COUNT, AVG, SUM

The COUNT(), AVG(), and SUM() functions are used for aggregate calculations...

SELECT COUNT(*) AS TotalStudents, AVG(Age) AS AverageAge, SUM(Age) AS TotalAge FROM Students;

16. MySQL LIKE

The LIKE operator is used to search for a specified pattern in a column...

SELECT * FROM Students WHERE StudentName LIKE 'A%';

17. MySQL Wildcards

Wildcards are used with the LIKE operator to substitute one or more characters...

SELECT * FROM Students WHERE StudentName LIKE '_l%';

18. MySQL IN

The IN operator allows you to specify multiple values in a WHERE clause...

SELECT * FROM Students WHERE Age IN (20, 22, 24);

19. MySQL BETWEEN

The BETWEEN operator selects values within a range...

SELECT * FROM Students WHERE Age BETWEEN 20 AND 25;

20. MySQL Aliases

Aliases are used to give a table or a column a temporary name...

SELECT StudentName AS Name, Age AS Years FROM Students;

21. MySQL Joins

Joins are used to combine rows from two or more tables...

22. MySQL INNER JOIN

The INNER JOIN selects records that have matching values in both tables...

SELECT Students.StudentName, Courses.CourseName FROM Students INNER JOIN Courses ON Students.CourseID = Courses.CourseID;

23. MySQL LEFT JOIN

The LEFT JOIN returns all records from the left table, and the matched records from the right table...

SELECT Students.StudentName, Courses.CourseName FROM Students LEFT JOIN Courses ON Students.CourseID = Courses.CourseID;

24. MySQL RIGHT JOIN

The RIGHT JOIN returns all records from the right table, and the matched records from the left table...

SELECT Students.StudentName, Courses.CourseName FROM Students RIGHT JOIN Courses ON Students.CourseID = Courses.CourseID;

25. MySQL CROSS JOIN

The CROSS JOIN returns the Cartesian product of two tables...

SELECT Students.StudentName, Courses.CourseName FROM Students CROSS JOIN Courses;

26. MySQL Self Join

A self join is a regular join but the table is joined with itself...

SELECT A.StudentName AS Student1, B.StudentName AS Student2 FROM Students A, Students B WHERE A.StudentID <> B.StudentID;

28. MySQL Database

A MySQL database is used to store data in tables...

29. MySQL Create DB

The CREATE DATABASE statement is used to create a new database...

CREATE DATABASE SchoolDB;

30. MySQL Drop DB

The DROP DATABASE statement is used to delete a database...

DROP DATABASE SchoolDB;

31. MySQL Create Table

The CREATE TABLE statement is used to create a new table in the database...

CREATE TABLE Students ( StudentID INT AUTO_INCREMENT, StudentName VARCHAR(255), Age INT, PRIMARY KEY (StudentID) );

32. MySQL Drop Table

The DROP TABLE statement is used to delete an existing table...

DROP TABLE Students;

33. MySQL Alter Table

The ALTER TABLE statement is used to add, delete, or modify columns in a table...

ALTER TABLE Students ADD Gender VARCHAR(10);

34. MySQL Constraints

Constraints are used to specify rules for data in a table...

35. MySQL Not Null

The NOT NULL constraint ensures that a column cannot have a NULL value...

CREATE TABLE Students ( StudentID INT NOT NULL, StudentName VARCHAR(255) NOT NULL );

36. MySQL Unique

The UNIQUE constraint ensures that all values in a column are different...

CREATE TABLE Students ( Email VARCHAR(255) UNIQUE );

37. MySQL Primary Key

The PRIMARY KEY uniquely identifies each record in a table...

CREATE TABLE Students ( StudentID INT PRIMARY KEY );

38. MySQL Foreign Key

The FOREIGN KEY is used to link two tables together...

CREATE TABLE Orders ( OrderID INT PRIMARY KEY, StudentID INT, FOREIGN KEY (StudentID) REFERENCES Students(StudentID) );

39. MySQL Check

The CHECK constraint ensures that a column meets a specific condition...

CREATE TABLE Students ( Age INT CHECK (Age >= 18) );

40. MySQL Default

The DEFAULT constraint provides a default value for a column when none is specified...

CREATE TABLE Students ( EnrollmentStatus VARCHAR(50) DEFAULT 'Active' );

41. MySQL Create Index

The CREATE INDEX statement is used to create an index in a table...

CREATE INDEX idx_studentname ON Students(StudentName);

42. MySQL Auto Increment

The AUTO_INCREMENT attribute allows a unique number to be generated automatically for new records...

CREATE TABLE Students ( StudentID INT AUTO_INCREMENT PRIMARY KEY );

43. MySQL Dates

MySQL provides various functions to work with date and time values...

SELECT CURDATE(), NOW();

44. MySQL Views

A view is a virtual table based on the result-set of an SQL statement...

CREATE VIEW StudentView AS SELECT StudentName, Age FROM Students;

45. MySQL References

References are used to define relationships between tables...

46. MySQL Data Types

MySQL supports various data types for storing different kinds of data...

CREATE TABLE Students ( StudentID INT, StudentName VARCHAR(255), EnrollmentDate DATE );
